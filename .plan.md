# CyreneNameRoller 项目改造计划

## 项目概述
将纯静态 HTML/CSS/JS 的「随机点名器 v2.0.5」重构为 Vue 3 + Electron 桌面应用，UI 采用 Windows 11 Fluent Design 风格。

## 关键决策
- **两个模式都迁移**：随机点名器 + 翻牌抽名器
- **Tunnel 主题**：不保留
- **数据存储**：改用文件系统（通过 Electron 主进程读写 JSON）
- **品牌色**：保持粉色 #ea5ec1
- **路由结构**：左侧 Dock 导航栏，每个功能独立页面
- **许可证**：GPLv3

### electron/main.js 骨架
```js
const { app, BrowserWindow, ipcMain } = require('electron')
const path = require('path')
const DataManager = require('./dataManager')

let win

function createWindow() {
  win = new BrowserWindow({
    width: 1200,
    height: 800,
    frame: false,
    backgroundColor: '#00000000',
    webPreferences: {
      preload: path.join(__dirname, 'preload.js'),
      contextIsolation: true,
      nodeIntegration: false
    }
  })

  if (process.env.VITE_DEV_SERVER_URL) {
    win.loadURL(process.env.VITE_DEV_SERVER_URL)
  } else {
    win.loadFile(path.join(__dirname, '../dist/index.html'))
  }
}

// IPC: 窗口控制
ipcMain.on('window-minimize', () => win.minimize())
ipcMain.on('window-maximize', () => {
  win.isMaximized() ? win.unmaximize() : win.maximize()
})
ipcMain.on('window-close', () => win.close())

// IPC: 数据操作
ipcMain.handle('data:load', (_, key) => DataManager.load(key))
ipcMain.handle('data:save', (_, key, data) => DataManager.save(key, data))
ipcMain.handle('data:loadNames', () => DataManager.loadDefaultNames())
ipcMain.handle('data:exportData', () => DataManager.exportAll())
ipcMain.handle('data:importData', (_, jsonData) => DataManager.importAll(jsonData))

app.whenReady().then(createWindow)
```

### electron/preload.js
```js
const { contextBridge, ipcRenderer } = require('electron')

contextBridge.exposeInMainWorld('electronAPI', {
  minimize: () => ipcRenderer.send('window-minimize'),
  maximize: () => ipcRenderer.send('window-maximize'),
  close: () => ipcRenderer.send('window-close'),
  loadData: (key) => ipcRenderer.invoke('data:load', key),
  saveData: (key, data) => ipcRenderer.invoke('data:save', key, data),
  loadNames: () => ipcRenderer.invoke('data:loadNames'),
  exportData: () => ipcRenderer.invoke('data:exportData'),
  importData: (jsonData) => ipcRenderer.invoke('data:importData', jsonData)
})
```

### electron/dataManager.js
负责在用户数据目录下读写 JSON 文件：
- `settings.json` - 用户设置
- `lists.json` - 名单数据
- `statistics.json` - 统计数据
- `balance.json` - 平衡算法配置

## Step 4: 业务页面迁移

### 4.1 RollerView.vue（随机点名页）
从 `index.html` 迁移：
- 名字显示区域（带动画效果：entering, shaking, pulse, final）
- 控制面板：名单选择器、English Mode 开关、多人模式开关、人数设置
- 开始/停止按钮（带进度条）
- 平衡算法逻辑提取到 `utils/balance.js`
- 所有 FontAwesome 图标替换为 Fluent Icons
- 所有 onclick 改为 Vue 事件
- 数据操作通过 `dataBridge.js` 调用 Electron IPC

### 4.2 CardView.vue（翻牌点名页）
从 `tick/index.html` 迁移：
- 牌子网格（CSS Grid + 3D 翻转动画）
- 发牌动画（卡牌从随机方向飞入）
- 翻牌放大效果
- 收牌区域（历史记录）
- 洗牌/重置按钮

### 4.3 StatisticsView.vue（统计页）
- 名字统计列表（中英文、抽取次数、概率、平衡概率）
- 总计次数
- 数据可视化（可选）

### 4.4 ListsView.vue（名单管理页）
- 名单列表（创建/切换/删除）
- 人员列表（添加/编辑/删除/批量操作）
- 白名单管理
- 导入/导出 .cyrene 格式

### 4.5 SettingsView.vue（设置页）
- 主题设置（Light/Dark 切换）
- 性能设置（粒子/模糊/动画速度 - 保留这些设置项但去掉 Tunnel 相关）
- 平衡算法设置（含曲线预览 Canvas）
- 数据管理（清除/导出/导入）
- 缓存管理
- 更新日志（从 up.json 加载）
- 版本信息

### 4.6 AboutView.vue（关于页）
- 应用名称与版本
- 作者信息（Cyrene）
- 版权声明
- GPLv3 许可证说明
- ICP 备案号
- 项目链接

每一步完成后等待用户确认再进行下一步。
